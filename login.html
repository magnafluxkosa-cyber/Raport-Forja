Pregătit pentru autentificare.
Login-ul este făcut prin Supabase Auth, iar pagina păstrează configurația comună a proiectului. Pentru conturile create de tine, ID-ul este convertit automat în email tehnic de forma id@forja.local, dacă nu introduci direct un email.

Flux recomandat

1. Creezi contul în Supabase cu email tehnic unic și parolă setată de tine.

2. Utilizatorul primește doar ID-ul și parola.

3. După login, pagina redirecționează în index.html.

Exemple de ID

OP001 → op001@forja.local

VIEW001 → view001@forja.local

ADMIN poate rămâne alias pentru contul principal actual.
